WSO2 Application Server-M1 Release Notes

29 Mar 2016

Welcome to WSO2 Application Server, the successor of WSO2 Carbon based Application Server. WSO2 Application Server 6.0.0 is a complete revamp and is based on vanilla Apache Tomcat. WSO2 provides a number of features by means of extensions to Tomcat to add/enhance the functionality. It provides first class support for generic web applications and JAX-RS/JAX-WS web applications. The performance of the server and individual application can be monitored by integrating WSO2 Application Server with WSO2 Data Analytics Server.

WSO2 Application Server is an open source project and it is available under the Apache Software License (v2.0) .

Key Features

  • HTTP Statistics Monitoring
  • Webapp classloading runtimes
